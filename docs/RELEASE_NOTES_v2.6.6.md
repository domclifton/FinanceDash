# InvestHome v2.6.6 Release Notes

This release cleans up the Trading 212 credentials UI so the app now presents a single credentials workflow through Settings.

## Changes
- Removed `.env` fallback wording from the Trading 212 settings page.
- Removed the Credential Source status from Settings.
- Updated Trading 212 status text to refer only to Settings credentials.
- Standardised Trading 212 credential lookup on the app settings database.
- Updated version, changelog, README, and install notes.
