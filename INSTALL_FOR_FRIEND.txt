INVESTHOME FINANCE TRACKER v2.6.9 - EASY INSTALL
=================================================

GitHub install:

   git clone https://github.com/domclifton/InvestHome.git
   cd InvestHome
   chmod +x scripts/install.sh scripts/start.sh scripts/backup_db.sh
   ./scripts/install.sh
   ./scripts/start.sh

Manual install:

   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   cp .env.example .env
   python3 app.py

Open it in a browser:

   http://SERVER-IP:5000

TRADING 212 OPTIONAL SETUP
==========================

Optional: add Trading 212 credentials in Settings → Trading 212 Connection after first launch.
Credentials are saved in the application settings and can be overwritten from the same page.

UPDATING FROM AN OLDER VERSION
==============================

Keep these files:

   finance.db
   .env

Before updating:

   ./scripts/backup_db.sh

Then overwrite/pull the app files and restart.

RESETTING THE APP
=================

To start from fresh data:

   rm finance.db
   python3 app.py

DO NOT COMMIT
=============

Do not commit:

   .env
   finance.db

These are already ignored by .gitignore.
